ইউটিউব ভিডিও কিভাবে আমাদের ওয়েবসাইটে ব্যবহার করবো?

ধাপ ১ঃ- প্রথমে যেকোন একটি ব্রাউজার ওপেন করে YouTube এ গিয়ে যেকোন একটা ভিডিও চালু করুন

ধাপ ২ঃ-তারপর যেকোন ভিডিওর নিচে Share এ Click করে Embed> এ ক্লিক করলে কিছু কোড আপনার সামনে চলে আসবে এখান থেকে কোড গুলো কপি করে আপনার ওয়েবসাইটের body tag এর ভিতর যেকোন অংশে কোডটি পেস্ট করে দিতে পারলেই আপনি চ্যাম্পিয়ন ।

Note: অনেক সময় অনেক চ্যানেল তাদের ভিডিও অন্য কোন ওয়েবসাইটে ব্যবহার করার অনুমতি দেয়না, সেক্ষেত্রে ভয় না পেয়ে অন্য একটি ভিডিও দিয়ে চেস্টা করুন


অন্য একটি ওয়েবসাইট কিভাবে আমাদের ওয়েবসাইটে ব্যবহার করবো?

ধাপ ১ঃ- প্রথমে iframe tag নিতে হবে। তারপর যেকোন একটি ব্রাউজার ওপেন করে যেকোন একটি ওয়েবসাইটের লিংক কপি করুন

ধাপ ২ঃ- তারপর iframe tag এর ভিতর src এট্রিবিউট নিয়ে তার ভিতর কপি করা লিংকটি বসিয়ে দিন

ধাপ ৩ঃ- আর একটু কাজ করতে হবে আপনার, src এর মতো আরো ২টা এট্রিবিউট নিতে হবে, যথাক্রমে width এবং height ও এর ভিতর আপনার যতটুকু ইচ্ছা height এবং width সেট করে দিলেই কেল্লাফতে!

প্রশ্নঃ- Blockline element কি?

উত্তরঃ- যেই ট্যাগ গুলো ব্যবহার করলে ওয়েবসাইটের সম্পূর্ণ জায়গা দখল করে থাকে তাদের ব্লকলাইন এলিমেন্ট বলে ।

প্রশ্নঃ- Inline element কি?

উত্তরঃ- যেই ট্যাগ গুলো ব্যবহার করলে ওয়েবসাইটের সম্পূর্ণ জায়গা দখল না করে তার যতটুকু প্রয়োজন ততটুকু দখল করে তাদের ইনলাইন এলিমেন্ট বলে ।

i tag i ট্যাগ এর মাধ্যমে আমরা কোন টেক্সটকে italic ট্যাক্সট এ রুপান্তর করতে পারি।
br tag লাইন ব্রেক দেওয়া জন্য br tag ব্যবহ্রত হয়।
style attribute কোন লিখার কালার পরিবর্তন করতে style attribut ব্যবহার করতে হয়। (সিএস এস সেখার সময় আরো ভালো করে জানবো)



Input TAG

Input type text যেকোন ধরনের নাম অথবা টেক্সট ইনপুট হিসাবে নেওয়ার জন্য এটি ব্যবহ্রত হয়
Input type file যেকোন ধরনের file অথবা image ইনপুট হিসাবে নেওয়ার জন্য এটি ব্যবহ্রত হয়
Input type number যেকোন ধরনের নাম্বার ইনপুট হিসাবে নেওয়ার জন্য এটি ব্যবহ্রত হয়
Input type email E-mail ইনপুট হিসাবে নেওয়ার জন্য এটি ব্যবহ্রত হয়
Input type date তারিখ ইনপুট হিসাবে নেওয়ার জন্য এটি ব্যবহ্রত হয়
Input type radio এটি সাধারনত Gender ইনপুট হিসাবে নেওয়ার জন্য ব্যবহ্রত হয়ে থাকে।

select এবং option TAG কোন একটা ফিল্ডের উপর ক্লিক করলে অনেক অপশন চলে আসবে । এটা করার ক্ষেত্রে এটি ব্যবহ্রত করা হয়ে থাকে
